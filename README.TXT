- To run the program, the user needs to call the main method on
the Controller class and specify the name of the input file and the excluded list file (by default "inputFile.txt" and "excludedList.txt".
- Once the program runs, the user will be asked to enter Y
to start a search. Any other key entered will exit the program
- After that, the user will be asked to enter keywords to search
against the database. 
- There are several connectors and signs user can use
	- and: to return results that match both clauses
	- or: to return results that match at least one of two
		clauses
	- "-": to return results that do not contain the word

- The top 5 results for each query will query will then be
presented
